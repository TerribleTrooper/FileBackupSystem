Описание:

FileBackupSystem — это кроссплатформенное консольное приложение,
которое выполняет резервное копирование файлов из одной или нескольких
исходных папок в целевую директорию.

При каждом запуске:
- создаётся папка с временным штампом;
- копируются все доступные файлы;
- процесс журналируется;
- результат архивируется в ZIP-файл.

Основные возможности:
- Поддержка нескольких исходных папок
- Создание папки резервной копии с timestamp
- Корректная обработка ошибок доступа к файлам
- Логирование процесса копирования:
    - Error
    - Info
    - Debug
- Отдельный файл лога для каждого запуска
- Архивирование результата в ZIP
- Кроссплатформенность (Windows / macOS / Linux)

Используемые технологии:
- C#
- .NET 10
- System.IO
- System.Text.Json
- System.IO.Compression
